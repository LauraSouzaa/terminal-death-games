# 🎮 Terminal Death Games

> Um jogo interativo em Python baseado em múltiplas fases de sobrevivência, onde lógica, memória e raciocínio rápido são essenciais para vencer.

---

## 🧠 Conceito

Inspirado em jogos de sobrevivência psicológica, o jogador enfrenta desafios progressivos que testam suas habilidades sob pressão. Cada erro pode significar o fim da partida.

---

## 🎯 Objetivo

Sobreviver a todas as fases completando corretamente os desafios propostos e avançar até o final do jogo.

---

## 🚀 Funcionalidades

* 🔹 Sistema de fases progressivas
* 🔹 Desafios matemáticos dinâmicos
* 🔹 Testes de memória
* 🔹 Narrativa imersiva no terminal
* 🔹 Sistema de reinício do jogo

---

## 🛠 Tecnologias utilizadas

* Python 3
* Biblioteca `random`
* Biblioteca `time`
* Biblioteca `os`

---

## 📦 Instalação e execução

### 1. Clone o repositório

```bash
git clone https://github.com/LauraSouzaa/terminal-death-games.git
```

### 2. Acesse a pasta do projeto

```bash
cd terminal-death-games
```

### 3. Execute o jogo

```bash
python main.py
```

---

## 🎮 Como jogar

* Leia atentamente as instruções exibidas no terminal
* Responda corretamente aos desafios
* Use lógica e memória para avançar
* Evite erros — eles podem encerrar o jogo
