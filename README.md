# 🎮 Terminal Death Games

Um jogo interativo em Python baseado em múltiplas fases de sobrevivência.
Cada fase testa habilidades como lógica, matemática e memória sob pressão.

## Conceito
Inspirado em jogos de sobrevivência psicológica, o jogador precisa tomar decisões e resolver desafios para avançar.

** Objetivo
Sobreviver a todas as fases acertando os desafios propostos.

## 🚀 Funcionalidades
- Sistema de fases progressivas
- Desafios matemáticos dinâmicos
- Teste de memória
- Narrativa imersiva no terminal
- Sistema de reinício do jogo

## 🛠 Tecnologias utilizadas
- Python 3
- Biblioteca `random`
- Biblioteca `time`
- Biblioteca `os`

## ▶️ Como executar

python main.py
